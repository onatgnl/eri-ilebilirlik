# Erişilebilirlik Widget'ı

Bu widget, web sitelerine ve local projelere kolayca entegre edilebilen bir erişilebilirlik menüsüdür.

## Kurulum

1. `public/eri-ilebilirlik-widget.js` ve `public/eri-ilebilirlik-widget.css` dosyalarını projenize ekleyin.
2. HTML dosyanıza aşağıdaki kodu ekleyin:

```html
<link rel="stylesheet" href="public/eri-ilebilirlik-widget.css">
<script src="public/eri-ilebilirlik-widget.js"></script>
```

## Özellikler
- Sağ altta aç/kapa butonu
- Ekran görüntüsündeki tüm erişilebilirlik özellikleri
- Her özellik için işlev eklenebilir

## Geliştirme
Her butonun işlevini `eri-ilebilirlik-widget.js` dosyasına ekleyebilirsiniz.
